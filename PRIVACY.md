# Privacy Policy

Ultimo aggiornamento: 28/06/2026

## 1. Dati raccolti
Il bot MCLink raccoglie e memorizza:
- **ID Discord** dell'utente
- **Username e UUID Minecraft** collegato tramite `/link`
- **Codici di verifica** temporanei (rimossi dopo 5 minuti)

## 2. Finalità
I dati vengono utilizzati esclusivamente per:
- Collegare account Discord a account Minecraft
- Verificare la proprietà dell'account Minecraft tramite comando `/verify` in-game
- Mostrare il profilo collegato tramite `/profile`

## 3. Condivisione dei dati
I dati non vengono condivisi con terze parti. Le richieste API sono effettuate solo verso:
- Mojang API (per verificare l'esistenza dell'account Minecraft)
- Crafatar (per le immagini della skin)

## 4. Conservazione
I dati vengono conservati fino a quando l'utente non esegue `/unlink`. I codici di verifica vengono eliminati dopo 5 minuti o al consumo.

## 5. Contatto
Per richiedere la cancellazione dei dati, contatta l'amministratore del server Discord.

## 6. Modifiche
Questa privacy policy può essere aggiornata in qualsiasi momento. Le modifiche saranno comunicate nel server Discord.
